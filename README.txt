##### Student score board
This application allows a user to add student data (name and score), view data of all students of a class.
User can also see summary of class

##### Run Application
To see application open index.html in your browser

##### Test cases 
To run test cases you need to first install node.js and its pacakage manager (npm). You can get them from 'http://nodejs.org/'.
After that run the following commands to setup test environment:

1. npm install -g karma-cli
2. npm install --save-dev karma karma-mocha karma-chai karma-chrome-launcher
3. npm install --save angular
4. npm install --save-dev angular-mocks
5. npm install karma-coverage --save-dev

##### To run test cases
After installing these packages run 'karma start' to run the test cases
